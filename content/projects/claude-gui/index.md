Multi-window Electron cockpit for the Claude Code CLI. Shows files touched with deep links into diffs, the agents in use and their respective prompts, tool calls grouped by purpose, and errors surfaced (rejected tool calls, denied permissions). It works alongside the retro-claude engine so you can navigate and control Claude Code from one surface.

The aim is to reduce the cognitive overhead of running a swarm of subagents in parallel — improving visibility and reducing the operational surface a single human has to keep in their head.
