# Runbooks — Index

Runbooks are procedural guides — step-by-step instructions for operational tasks like setup, reset, deployment, environment bootstrapping. They are **not** evergreen practices (no principles, no anti-patterns) and **not** feature specs (no behavior description). Just "how do I do this thing."

See `docs/doc-system/02-style-atlas.md §11` for the runbook style guide.

## How to add a new runbook

1. Create `docs/runbooks/<procedure-slug>.md` from `_template.md`.
2. Set `kind: runbook` in frontmatter (mandatory).
3. Add a row to the table below.
